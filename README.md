# AI 광고 제작 프로젝트

## 1. 프로젝트 개요

본 프로젝트는 생성형 AI를 활용하여 광고 영상을 제작하는 프로젝트입니다.

AI를 활용하여 광고 기획, 스토리보드 제작, 프롬프트 작성, 이미지 생성, 영상 제작 및 편집까지 전체 제작 과정을 진행했습니다.

생성형 AI를 단순히 결과물 생성에 사용하는 것이 아니라, 기획부터 최종 결과물 완성까지의 AI 기반 광고 제작 파이프라인을 구축하는 것을 목표로 했습니다.

# 2. 프로젝트 주제

## 생성형 AI 기반 광고 영상 제작

제품의 특징과 이미지를 효과적으로 전달하기 위해 생성형 AI 기술을 활용하여 광고 영상을 제작했습니다.

전체 과정은 아래 순서로 진행했습니다.

기획 → 스토리보드 제작 → 프롬프트 작성 → AI 이미지 생성 → AI 영상 제작 → 편집 및 최종 결과물 완성

### 크레딧 부족 상황 대응 전략

AI 생성 도구의 사용량 제한 또는 크레딧 부족 상황을 고려하여 아래와 같은 대체 전략을 수립하였다.

#### 1. 씬 축소 전략

- 기존 5개 씬 구성을 핵심 메시지 중심으로 3개 씬으로 축소
- 제품 소개, 제품 특징, 최종 브랜드 강조 장면 중심으로 우선 제작
- 불필요한 장면 생성을 줄여 AI 생성 비용 절감

#### 2. 저비용 도구 활용 전략

- 이미지 생성은 무료 사용 가능한 AI 도구 또는 기존 생성 이미지 재활용
- 영상 생성 횟수를 줄이고 정지 이미지를 활용한 영상 효과 적용
- CapCut의 무료 편집 기능을 활용하여 추가 비용 없이 완성

#### 3. 편집 전략 변경

- AI 영상 생성 대신 생성된 이미지를 활용한 줌인, 이동 효과 적용
- 반복 생성 대신 기존 결과물을 편집하여 품질 개선
- 짧은 광고 영상 형식으로 구성하여 제작 시간과 비용 절감

이를 통해 제한된 AI 크레딧 환경에서도 핵심 광고 메시지를 유지하면서 최종 결과물을 완성할 수 있도록 계획하였다.

| 도구 | 예상 품질 | 예상 속도 | 예상 비용 | 특징 |

| Runway | 높음 | 보통 | 유료 중심 | 자연스러운 움직임과 광고 영상 제작에 적합 |
| Pika | 보통~높음 | 빠름 | 무료/유료 혼합 | 짧은 영상 생성과 빠른 테스트에 적합 |
| CapCut AI 영상 기능 | 보통 | 빠름 | 무료 기능 활용 가능 | 편집과 영상 제작을 동시에 진행 가능 |

### 도구 선택 근거

- 고품질 영상 제작이 필요한 경우 Runway와 같은 전문 영상 생성 도구 활용 가능

- 최종 편집 및 완성 과정에서는 CapCut을 활용하여 제작 시간을 단축하였다

## T2I(Text-to-Image)와 I2V(Image-to-Video) 선택 근거

본 프로젝트에서는 광고 제작 목적에 맞추어 T2I(Text-to-Image)와 I2V(Image-to-Video) 방식을 함께 활용하였다.

| 구분 | T2I (Text-to-Image) | I2V (Image-to-Video) |
|---|---|---|
| 특징 | 텍스트를 기반으로 이미지를 생성 | 생성된 이미지를 기반으로 영상을 생성 |
| 장점 | 원하는 장면과 분위기를 세밀하게 표현 가능 | 자연스러운 움직임과 장면 전환 구현 가능 |
| 단점 | 움직임 표현이 어려움 | 입력 이미지 품질에 결과가 영향을 받음 |

### 본 프로젝트의 선택 근거

- 광고 장면의 핵심 이미지는 T2I(ChatGPT Image Generation)를 활용하여 제작하였다.
- 생성된 이미지와 장면을 기반으로 I2V(AI 영상 생성 도구)를 활용하여 자연스러운 영상 효과를 구현하였다.
- 정적인 이미지 품질과 영상의 자연스러운 움직임을 함께 확보하기 위해 두 방식을 병행하여 사용하였다.

## AI 생성 이미지 및 영상 활용

본 프로젝트에서는 AI 이미지 생성 및 AI 영상 생성 도구를 활용하여 시각 자료를 제작하였다.

직접 촬영한 사진 및 영상은 사용하지 않았으며, 유료 스톡 이미지·영상 또한 사용하지 않았다.

모든 이미지와 영상 결과물은 생성형 AI 도구를 통해 제작한 후 편집 과정에서 활용하였다.


## AI 생성 오디오 활용

본 프로젝트에서는 영상 제작 과정에서 AI 생성 오디오를 활용하였다.

- 오디오 생성 도구: CapCut AI Voice
- 활용 목적: 영상 내 설명 음성 및 배경 음향 제작
- 생성 파일: 프로젝트 소개 영상 내 AI 음성 파일 적용
- 활용 방식: AI가 생성한 음성을 영상 편집 과정에서 삽입하여 콘텐츠 전달력을 향상시킴

CapCut AI Voice 기능을 활용하여 별도의 직접 녹음 없이 프로젝트 설명 내용을 기반으로 AI 음성을 생성하였으며, 생성된 음성 파일을 최종 영상 제작에 활용하였다.

## 6. 장면 구성

### 영상 구성

| 씬 | 내용 | 활용 AI |
|---|---|---|
| Scene 1 | Almond Breeze 제품 소개 | ChatGPT Image Generation |
| Scene 2 | 제품 특징(고소한 맛, 식물성 음료) 표현 | ChatGPT Image Generation |
| Scene 3 | 건강한 라이프스타일 연출 | AI 영상 생성 도구 |
| Scene 4 | 제품 클로즈업 및 브랜드 강조 | AI 영상 생성 도구 |
| Scene 5 | 브랜드 로고 및 마무리 화면 | CapCut |

### 제작 계획

- 총 예상 씬 수: 5개
- 이미지 생성 시도 횟수: 2회
- 영상 생성 시도 횟수: 2회
- 최종 선정 버전: 2차 생성 결과
- 대체 이미지 생성 도구: DALL·E, Adobe Firefly
- 대체 영상 생성 도구: Pika, Runway

### 시간축 단축 시 장면 유지/삭제 기준

광고 영상 길이가 제한되는 경우 핵심 메시지 전달을 우선하여 장면을 조정한다.

#### 장면 유지 기준

- 제품 특징 전달에 직접적으로 필요한 장면 유지
- 브랜드 메시지와 제품 USP를 표현하는 장면 유지
- 시청자가 제품을 인식하는 데 필요한 제품 중심 장면 유지

#### 장면 삭제 기준

- 메시지 전달 효과가 낮은 보조 장면 삭제
- 유사한 분위기를 반복하는 장면 축소
- 제품 정보와 직접적인 관련이 적은 연출 장면 삭제

#### 짧은 버전 재구성 전략

기존 5개 씬 구성:

1. 제품 소개
2. 제품 특징 표현
3. 건강한 라이프스타일 연출
4. 제품 클로즈업
5. 브랜드 마무리

↓

시간 단축 버전(3개 씬):

1. 제품 소개 + 브랜드 메시지 통합
2. 제품 특징 및 라이프스타일 표현 통합
3. 제품 클로즈업 + 최종 브랜드 강조

짧은 영상에서는 제품 인식 → 핵심 가치 전달 → 브랜드 기억 형성 순서로 메시지를 재구성한다.

# 8. 프롬프트 개선 과정 (Pre / Post)

## Prompt A (개선 전)

Almond Breeze 광고 이미지 만들어줘.

제품이 잘 보이고 깔끔한 느낌으로 제작해줘.


### 문제점

- 제품 특징에 대한 구체적인 설명 부족
- 광고 분위기와 스타일 지정 부족
- 카메라 구도 및 조명 조건 없음
- 원하는 브랜드 이미지 표현이 어려움


---

## Prompt B (개선 후)

Almond Breeze 제품 광고 이미지 제작.

식물성 아몬드 음료의 건강하고 가벼운 이미지를 표현한다.

제품은 화면 중앙에 배치하고 고급스럽고 깨끗한 브랜드 광고 스타일로 제작한다.

자연스러운 조명과 현실적인 제품 질감을 표현하며,
프리미엄 음료 광고와 같은 전문적인 구도를 적용한다.

카메라 구도:
제품 중심 클로즈업

분위기:
밝고 건강한 라이프스타일

스타일:
실제 광고 촬영 이미지와 같은 현실적인 표현


### 개선 내용

Prompt A에서 부족했던 요소를 Prompt B에 추가하였다.

변경 사항:

- 제품 특징 추가 → 브랜드 메시지 명확화
- 광고 스타일 추가 → 결과물 분위기 개선
- 카메라 구도 추가 → 장면 표현 구체화
- 조명 조건 추가 → 이미지 품질 향상
- 분위기 설정 추가 → 브랜드 일관성 확보
# 9. 에셋 규격

## 이미지

- 파일 형식 : PNG / JPG
- 비율 : 16:9
- 해상도 : 1920 × 1080


## 영상

- 파일 형식 : MP4
- 해상도 : 720p
- 영상 길이 : 약 10초
- 프레임 : 30fps

  
## 색감 규격

영상의 색감은 전체 장면에서 일관성을 유지하기 위해 아래 기준을 적용하였다.

- 색공간(Color Space): sRGB
- 화이트밸런스: 자연광 기준(약 5500K)
- 컬러 톤: 밝고 깨끗한 광고 스타일
- LUT 적용: 별도 LUT 미사용
- 색감 기준: 모든 씬에서 동일한 밝기와 채도를 유지하여 브랜드 이미지 일관성을 확보
- 색보정: CapCut 기본 색보정 기능을 활용하여 밝기와 대비를 일관되게 조정

# 11. 파일명 규칙 및 버전 관리

## 파일명 규칙

프로젝트 결과물은 아래 규칙에 따라 파일명을 관리하였다.

파일명 형식

프로젝트명_콘텐츠명_버전.확장자

예시

- AlmondBreeze_Storyboard_v1.pdf
- AlmondBreeze_Image_Scene1_v1.png
- AlmondBreeze_Video_Scene3_v2.mp4
- AlmondBreeze_AI_Advertisement_v1.mp4

## 버전 관리 규칙

- v1 : 최초 생성 결과
- v2 : 프롬프트 수정 후 개선 결과
- v3 : 최종 검토 및 편집 완료 버전

## 재사용 가이드

- 동일한 파일명 규칙을 사용하여 다른 프로젝트에도 적용 가능
- 씬(Scene) 번호를 포함하여 장면별 결과물을 쉽게 관리
- 버전 번호(v1, v2, v3)를 통해 생성 결과와 수정 이력을 구분
