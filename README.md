# AI 광고 제작 프로젝트

## 1. 프로젝트 개요

본 프로젝트는 생성형 AI를 활용하여 광고 영상을 제작하는 프로젝트입니다.

AI를 활용하여 광고 기획, 스토리보드 제작, 프롬프트 작성, 이미지 생성, 영상 제작 및 편집까지 전체 제작 과정을 진행했습니다.

생성형 AI를 단순히 결과물 생성에 사용하는 것이 아니라, 기획부터 최종 결과물 완성까지의 AI 기반 광고 제작 파이프라인을 구축하는 것을 목표로 했습니다.

# 2. 프로젝트 주제

## 생성형 AI 기반 광고 영상 제작

제품의 특징과 이미지를 효과적으로 전달하기 위해 생성형 AI 기술을 활용하여 광고 영상을 제작했습니다.

전체 과정은 아래 순서로 진행했습니다.

기획 → 스토리보드 제작 → 프롬프트 작성 → AI 이미지 생성 → AI 영상 제작 → 편집 및 최종 결과물 완성

# 4. 제작 파이프라인

## Step 1. 광고 기획

- 제품 및 서비스 분석
- 광고 목적 설정
- 타깃 고객 설정
- 광고 메시지 및 콘셉트 기획

---

## Step 2. 스토리보드 제작

광고 영상의 전체적인 흐름을 설계하고 장면별 제작 방향을 정리했습니다.

제작 내용

- 장면 구성
- 화면 연출 방향 설정
- 카메라 구도 계획
- 영상 흐름 설계

---

## Step 3. 프롬프트 작성

AI가 원하는 결과물을 생성할 수 있도록 제품 특징, 분위기, 구도, 조명, 스타일을 포함한 상세 프롬프트를 작성했습니다.

프롬프트 작성 시 고려 요소

- 제품 특징
- 광고 분위기
- 이미지 스타일
- 카메라 구도
- 조명 표현
- 영상 움직임

---

## Step 4. AI 결과물 제작

AI 도구를 활용하여 광고 제작에 필요한 이미지와 영상을 생성했습니다.

제작 과정

1. 프롬프트 작성
2. AI 결과물 생성
3. 결과 확인
4. 부족한 부분 수정
5. 최종 결과물 선정

---

## Step 5. 영상 편집 및 완성

- AI 생성 결과물 배치
- 장면 연결
- 자막 추가
- 영상 효과 적용
- 최종 결과물 검수

# 6. 프롬프트 기록

## 이미지 생성 Prompt

제품 광고 이미지 제작.
고급스럽고 깨끗한 브랜드 광고 스타일.
제품이 화면 중심에 위치하고 자연스러운 조명 표현.
현실적인 제품 질감과 프리미엄한 분위기.
광고 촬영 이미지와 같은 전문적인 구도.


## 영상 생성 Prompt

제품을 강조하는 광고 영상.
부드러운 카메라 움직임과 자연스러운 장면 전환.
고급스럽고 세련된 브랜드 광고 분위기.
제품의 특징이 잘 드러나는 영상 연출.
실제 광고 촬영과 같은 현실적인 스타일.

# 7. 프롬프트 개선 과정 (Pre / Post)

## Pre Prompt

초기 프롬프트는 제품과 기본적인 방향만 작성했습니다.

문제점

- 분위기 표현 부족
- 장면 구성 부족
- 원하는 광고 스타일 표현 부족


## Post Prompt

결과물 품질 향상을 위해 구체적인 조건을 추가했습니다.

개선 내용

- 제품 위치 설정
- 배경 분위기 추가
- 조명 조건 추가
- 카메라 구도 설정
- 광고 스타일 구체화
