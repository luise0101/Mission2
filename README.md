# AI 광고 제작 프로젝트

## 1. 프로젝트 개요

본 프로젝트는 생성형 AI를 활용하여 광고 영상을 제작하는 프로젝트입니다.

AI를 활용하여 광고 기획, 스토리보드 제작, 프롬프트 작성, 이미지 생성, 영상 제작 및 편집까지 전체 제작 과정을 진행했습니다.

생성형 AI를 단순히 결과물 생성에 사용하는 것이 아니라, 기획부터 최종 결과물 완성까지의 AI 기반 광고 제작 파이프라인을 구축하는 것을 목표로 했습니다.

# 2. 프로젝트 주제

## 생성형 AI 기반 광고 영상 제작

제품의 특징과 이미지를 효과적으로 전달하기 위해 생성형 AI 기술을 활용하여 광고 영상을 제작했습니다.

전체 과정은 아래 순서로 진행했습니다.

기획 → 스토리보드 제작 → 프롬프트 작성 → AI 이미지 생성 → AI 영상 제작 → 편집 및 최종 결과물 완성

## AI 도구 선정 기준

본 프로젝트에서는 결과물의 품질, 제작 속도, 비용을 고려하여 AI 도구를 선정하였다.

| 도구 | 품질 | 속도 | 비용 | 선정 이유 |
|---|---|---|---|---|
| ChatGPT | 높음 | 빠름 | 무료/유료 선택 가능 | 광고 기획 및 프롬프트 작성에 적합 |
| ChatGPT Image Generation | 높음 | 빠름 | ChatGPT 기능 활용 | 광고용 이미지 품질이 우수하여 선택 |
| AI 영상 생성 도구 | 높음 | 보통 | 무료 체험 가능 | 이미지를 자연스러운 영상으로 변환 가능 |
| CapCut AI Voice | 보통~높음 | 빠름 | 무료 기능 활용 | 별도 녹음 없이 AI 음성 생성 가능 |
| CapCut | 높음 | 빠름 | 무료 기능 활용 | 자막, 효과, 색보정 등 후처리에 적합 |

## T2I(Text-to-Image)와 I2V(Image-to-Video) 선택 근거

본 프로젝트에서는 광고 제작 목적에 맞추어 T2I(Text-to-Image)와 I2V(Image-to-Video) 방식을 함께 활용하였다.

| 구분 | T2I (Text-to-Image) | I2V (Image-to-Video) |
|---|---|---|
| 특징 | 텍스트를 기반으로 이미지를 생성 | 생성된 이미지를 기반으로 영상을 생성 |
| 장점 | 원하는 장면과 분위기를 세밀하게 표현 가능 | 자연스러운 움직임과 장면 전환 구현 가능 |
| 단점 | 움직임 표현이 어려움 | 입력 이미지 품질에 결과가 영향을 받음 |

### 본 프로젝트의 선택 근거

- 광고 장면의 핵심 이미지는 T2I(ChatGPT Image Generation)를 활용하여 제작하였다.
- 생성된 이미지와 장면을 기반으로 I2V(AI 영상 생성 도구)를 활용하여 자연스러운 영상 효과를 구현하였다.
- 정적인 이미지 품질과 영상의 자연스러운 움직임을 함께 확보하기 위해 두 방식을 병행하여 사용하였다.

## AI 생성 이미지 및 영상 활용

본 프로젝트에서는 AI 이미지 생성 및 AI 영상 생성 도구를 활용하여 시각 자료를 제작하였다.

직접 촬영한 사진 및 영상은 사용하지 않았으며, 유료 스톡 이미지·영상 또한 사용하지 않았다.

모든 이미지와 영상 결과물은 생성형 AI 도구를 통해 제작한 후 편집 과정에서 활용하였다.


## AI 생성 오디오 활용

본 프로젝트에서는 영상 제작 과정에서 AI 생성 오디오를 활용하였다.

- 오디오 생성 도구: CapCut AI Voice
- 활용 목적: 영상 내 설명 음성 및 배경 음향 제작
- 생성 파일: 프로젝트 소개 영상 내 AI 음성 파일 적용
- 활용 방식: AI가 생성한 음성을 영상 편집 과정에서 삽입하여 콘텐츠 전달력을 향상시킴

CapCut AI Voice 기능을 활용하여 별도의 직접 녹음 없이 프로젝트 설명 내용을 기반으로 AI 음성을 생성하였으며, 생성된 음성 파일을 최종 영상 제작에 활용하였다.

# 9. 에셋 규격

## 이미지

- 파일 형식 : PNG / JPG
- 비율 : 16:9
- 해상도 : 1920 × 1080


## 영상

- 파일 형식 : MP4
- 해상도 : 720p
- 영상 길이 : 약 10초
- 프레임 : 30fps

  
## 색감 규격

영상의 색감은 전체 장면에서 일관성을 유지하기 위해 아래 기준을 적용하였다.

- 색공간(Color Space): sRGB
- 화이트밸런스: 자연광 기준(약 5500K)
- 컬러 톤: 밝고 깨끗한 광고 스타일
- LUT 적용: 별도 LUT 미사용
- 색보정: CapCut 기본 색보정 기능을 활용하여 밝기와 대비를 일관되게 조정
